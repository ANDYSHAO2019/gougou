# 公网测试地址

当前临时公网地址：

```text
https://green-eagles-jump.loca.lt/
```

用途：

- 给朋友打开这个 HTTPS 地址测试。
- 麦克风权限可以在 HTTPS 下使用。
- 联网对战仍使用页面内的“联网对战”连接码流程。

注意：

- 这个地址依赖本机当前的 `localtunnel` 进程。
- 电脑休眠、断网、关闭终端进程后地址会失效。
- 如果地址失效，重新运行隧道会生成一个新地址。
